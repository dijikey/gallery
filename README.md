### gallery
Encrypt and Decrypt files with password key

### Use

1. Run "gallery.exe"
2. Choose Encrypt/Decrypt
3. Enter key
4. Choose folder (native way)
5. Wait ( Encrypt is long process )
6. Find the folder where u choose folder

### Features

- Encrypt/Decrypt files with password
- File name is saved
- Folder name not saved (wip)
